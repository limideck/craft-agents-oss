# Component Spec (stub)

Guidance for Grose Sites agents implementing UI components.

## Goals

- Prefer small, composable React components under `src/`
- Keep props typed; avoid prop-drilling when local state suffices
- Meet basic a11y (labels, focus, keyboard) for interactive controls

## Stack notes

Use Tailwind utility classes already set up in the template. Do not introduce a second CSS system unless asked.
