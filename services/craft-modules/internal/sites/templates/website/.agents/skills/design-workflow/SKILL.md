# Design Workflow (stub)

Guidance for Craft Sites agents collaborating with the Chat | Files | Preview workbench.

## Suggested loop

1. Clarify goal and audience in chat
2. Edit source via tools / Files panel
3. Start or refresh Preview (Vite on ports 5400+)
4. Use visual-edit for small text tweaks; larger layout changes in source
5. Verify responsive behavior before finishing
